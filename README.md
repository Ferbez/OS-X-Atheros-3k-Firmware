## IOath3kfrmw, RehabMan fork

Atheros 3k (AR3011) kernel extension for Mac OS X - firmware upload.

Currently known to work on: Snow Leopard (10.6.8), Lion, Mountain Lion, Mavericks, Yosemite, and possibly future OS X.
